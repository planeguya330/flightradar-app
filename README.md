# FlightRadar Live

## Overview
FlightRadar Live is a real-time flight tracking application that provides users with up-to-date information about flights around the world. The application allows users to visualize flight paths, view aircraft details, and obtain current flight status.

## Features
- **Live Flight Tracking**: Monitor active flights in real time.
- **Aircraft Details**: View detailed information about each aircraft, including its model, origin, and destination.
- **Search Functionality**: Search for specific flights by flight number, airline, or airport.
- **Map Integration**: Interactive map showcasing real-time flight paths.
- **User-Friendly Interface**: Easy to navigate and use for all types of users.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/planeguya330/flightradar-app.git
   ```
2. Change into the application directory:
   ```bash
   cd flightradar-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
- Open your web browser and navigate to `http://localhost:3000`.
- Use the search bar to find specific flights or browse the map for real-time flight tracking.

## Technology Stack
- **Frontend**: React.js, Axios
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **APIs**: Public Flight Tracking APIs
